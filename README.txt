== Description ==

WooCommerce Unlimited Upsell Lite gives you the ability to offer an upsell product at the point of checkout based on the contents of the customers shopping cart.

###Plugin Requirements###

This plugin requires the WooCommerce plugin.[Get WooCommerce](https://wordpress.org/plugins/woocommerce/)

###Available in WooCommerce Unlimited Upsell Pro Only###

There is also a Pro version that contains the following additional options.

* **Price Range** - Make the upsell popup available when cart total is in a specific price range.
* **Date Range** - Make the upsell popup available between specific dates.
* **Category** - Make the upsell popup available when the items in cart are in a specific category.
* **Quantity** - Make the upsell popup available when the quantity of items in cart is in a specific range.

== Installation ==

There are three ways to install this plugin.

= 1. Admin Search =
1. NOTE: Make sure that you have the WP Simple Pay Lite plugin installed.
1. In your Admin, go to menu Plugins > Add.
1. Search for `WooCommerce Unlimited Upsell`.
1. Find the plugin that's labeled `WooCommerce Unlimited Upsell`.
1. Look for the author name `Kungwoo` on the plugin.
1. Click to install.
1. Activate the plugin.
1. A new menu item `WooCommerce Unlimited Upsell Lite` will appear in the WordPress menu.

= 2. Download & Upload =
1. Download the plugin (a zip file) on the right column of this page.
1. In your Admin, go to menu Plugins > Add.
1. Select the tab "Upload".
1. Upload the .zip file you just downloaded.
1. Activate the plugin.
1. A new menu item `WooCommerce Unlimited Upsell Lite` will appear in the WordPress menu.

= 3. FTP Upload =
1. Download the plugin (.zip file) on the right column of this page.
1. Unzip the zip file contents.
1. Upload the `WooCommerce Unlimited Upsell Lite` folder to the `/wp-content/plugins/` directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. A new menu item `WooCommerce Unlimited Upsell Lite` will appear in the WordPress menu.
